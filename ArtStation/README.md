## 环境：

+ Windows + python3.6及以上
  + reuqests
+ Linux + python3.6及以上
  + requests
+ Wifi
------

运行后只需要粘贴一系列你想下载画师的主页链接：

比如[这个](https://www.artstation.com/timbougami)
以星花号： * 作为停止。
然后会有英文提示  ~~当然你不识字我也没有办法~~

配合高速网络食用更佳。quq。
